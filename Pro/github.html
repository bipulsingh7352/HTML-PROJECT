Frontend development

Frontend development is the development of the grafhical user interface of a website,through the use of HTML,CSS and JavaScript, so that users can view and interact with that website.


  • HTML:The Hyper Markup Language or HTML is the standard markup language for documents designed to be display in a web brouser.


  • CSS:Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML
  1. Pure CSS
  2. Bootstrap (Framework)
  3. Tailwind (Framework)
  4. Sass (Preprocessor)

  • JavaScript:JavaScript often abbreviated JS, is a programming language that is one of the core technologies of the world wide web,alongside HTML and CSS.It is use to add functionality in the website.
  1. Vanilla JS
  2. jQuery(Libray)
  3. React(Framework)
  4. Angular(Framework)
  5. Vue(Framework)
  6. Typescript(Preprocessor)
